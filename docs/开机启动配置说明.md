# 开机启动配置说明

## `start.bat` 是做什么的？

`start.bat` 是项目的**交互式一键启动脚本**，主要完成以下工作：

1. 检查 Python / pip 环境
2. 自动创建并激活 `venv` 虚拟环境
3. 安装 `requirements.txt` 中的依赖
4. 检查或生成 `.env` 配置文件
5. 输出访问地址（管理后台、登录页、API 文档等）
6. 最后执行 `python app.py` 启动 FastAPI 服务

它适合**首次部署或手动启动**，但因为它会打开 `cmd /k` 窗口并需要交互，所以**不适合直接放到开机启动里**。

---

## 开机自启文件说明

项目中已生成以下用于开机自启的文件：

| 文件 | 作用 |
|------|------|
| `start_hidden.bat` | 后台运行服务，无弹窗，输出写入 `logs\startup.log` |
| `startup.vbs` | 隐藏窗口调用 `start_hidden.bat`，作为开机自启入口 |
| `setup_startup_task.bat` | 自动创建 Windows 计划任务（需管理员运行） |

---

## 方式一：启动文件夹（最简单）

### 配置开机启动

1. 按 `Win + R`，输入 `shell:startup`，回车。
2. 将项目根目录下的 `startup.vbs` 复制到打开的启动文件夹中。
3. 重启电脑，登录后服务会自动在后台启动。
4. 服务日志查看：`logs\startup.log`

### 取消开机启动

1. 按 `Win + R`，输入 `shell:startup`，回车。
2. 找到 `startup.vbs`，右键删除。

> 如果也复制到了公共启动文件夹，可按 `Win + R` 输入 `shell:common startup` 检查并删除。

---

## 方式二：Windows 计划任务（更稳定）

### 配置开机启动

1. 右键 `setup_startup_task.bat`，选择 **以管理员身份运行**。
2. 脚本会创建一个名为 `WeChatDownloadAPI_AutoStart` 的计划任务。
3. 下次登录 Windows 时将自动在后台启动服务。

### 取消开机启动

以管理员身份运行 CMD，执行：

```cmd
schtasks /Delete /TN WeChatDownloadAPI_AutoStart /F
```

或者在 **任务计划程序** 中找到 `WeChatDownloadAPI_AutoStart`，右键选择 **删除**。

---

## 停止当前正在运行的服务

取消开机启动后，如果服务当前仍在运行，可以：

- 打开 **任务管理器** → 找到 `python.exe` 或 `wscript.exe` 相关进程 → 结束任务。
- 或在 CMD 中执行：

```cmd
taskkill /F /IM python.exe
```

> 注意：`taskkill /F /IM python.exe` 会结束所有 Python 进程。如果电脑上还有其他 Python 程序在运行，建议到任务管理器中单独结束对应进程。

---

## 注意事项

- **首次部署请先用 `start.bat`**，完成环境配置、依赖安装和微信扫码登录。
- 开机自启前请确认 `.env` 中已有有效微信登录凭证，否则服务启动后仍需手动登录。
- 服务默认监听 `http://localhost:5000`。
